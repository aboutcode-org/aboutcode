ScanCode Workbench
==================

ScanCode Workbench is a desktop
application (based on Electron) to review the results of a scan and
document your conclusions about the origin and license of software
components and packages.

  - **Documentation**: https://scancode-workbench.readthedocs.io
  - **Source Code**: https://github.com/aboutcode-org/scancode-workbench
